bindChannel:failureResult:

连接云推送服务. 向push server的bind channel发送请求。 如果执行成功，bindResult包含app ID, user ID和Channel ID。 如果执行失败，failureResult包含失败原因。 

- (void)bindChannel:(FrontiaPushBindCallback)bindResult failureResult:(FrontiaPushActionFailureCallback)failureResult;
Parameters
bindResult

连接成功时的回调函数。接收的参数是云端返回的app ID、 user ID和Channel ID。

failureResult

连接失败时的回调函数。包含错误码和错误原因。

Return Value

无
